## Overview

The purpose of this project is to create a chatbot that can assist with customer inquiries at a call center using OpenAI and Streamlit. The target audience are staff members who handle customer queries, who interact with the bot through a chat interface and receive responses from the model. It includes features such as buttons for starting new conversations and clearing the chat history, sliders to adjust temperature and top P parameters, and select boxes to choose between different models for generating responses.

## Getting Started

**System Requirements:**

* **Python**: `version x.y.z`

### Installation

<h4>From <code>source</code></h4>

> 1. Clone the Buddy-X repository:
>
> ```console
> $ git clone https://github.com/Cosmin-T/Buddy-X.git
> ```
>
> 2. Change to the project directory:
>
> ```console
> $ cd Buddy-X
> ```
>
> 3. Install the dependencies:
>
> ```console
> $ pip install -r requirements.txt
> ```

### Usage

<h4>From <code>source</code></h4>

> Run Buddy-X using the command below:
>
> ```console
> $ python main.py
> ```

### Tests

> Run the test suite using the command below:
>
> ```console
> $ pytest
> ```
